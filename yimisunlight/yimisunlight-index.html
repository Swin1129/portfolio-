這裡，是被陽光祝福的地方──
一米陽光，妳正在尋找的幸福彼方。

讓每個少女一見傾心的法式浪漫鄉村空間
一米陽光不僅是主人送給旅人的禮物，
也是新娘指定拍婚紗的地方
處處佈滿可愛寫意的雜貨，
巧心佈置的角落像是雜誌般令人驚艷
三間房風格不盡相同，有各自的甜蜜浪漫
戶外陽光庭園與綠油油的草地，
最適合來個幸福又芬芳的野餐時光！

一米陽光以溫馨居家鄉村風格提供「場地、空間租借」服務
地點位在國立美術館旁，近勤美誠品綠園道，大面積自然的窗光，
適合婚紗攝影，孕寫真，親子全家福寫真，專業網拍及商業產品攝影。

場地、空間租借攝影費用：

  • 使用1樓空間，費用 800 元/H。
  • 使用1樓空間+任選1間房型+2樓小客廳，費用 1200 元/H。

注意事項:

  • 每組同行人數最多為六人( 攝影師最多兩人 )。
  • 基本租借時數為1小時。
  • 第2小時起，若拍攝時間超過10分鐘，則以半小時計算，超過半小時以1小時計算。
  • 提前預約勘景，確認預約需預付50%租金作為訂金。
  • 如需變更時間，請在三天前來電更改時間，如臨時取消恕無法退費。
  • 預約當日未前來使用，視同放棄使用權，恕無法退費及變更檔期。
  • 如因操作不當造成道具或場景損壞，需照價賠償。
  • 請勿攜帶寵物及禁止飲食，垃圾煩請自行清除帶走。

PS.若有其他問題都歡迎來電詢問，謝謝!!

一米陽光路線資訊:

開車的旅人:

國道一號 → 台中/中港交流道下(往台中火車站方向) → 台灣大道直開 → 右轉忠明南路 → 左轉存中街 → 抵達一米陽光。

搭大眾運輸的旅人:

  • 從台中高鐵站出發：搭乘高鐵快捷公車159號(往台中公園方向)，土庫停車場(國家美術館站)下車。
  • 從台中火車站出發：在台中火車站對面統聯客運門口站牌，搭乘75號公車至五權西忠明南路口下車。
  • 從朝馬出發：可搭75號公車至五權西忠明南路口下車
  • 從中港轉運站出發：可搭75號公車至五權西忠明南路口下車。

從機場的旅人:

  • 從松山機場出發：
    搭乘巴士 → 先搭捷運電車到台北車站，轉搭國光或統聯客運巴士至台中火車站站下車。
    搭乘高鐵 → 先搭捷運電車到台北車站，轉搭台灣高鐵至烏日站下車。
  • 從桃園機場出發：
    搭乘巴士 → 建議搭乘巴士前往台中最快最方便。在機場可直接搭乘統聯或國光客運巴士至台中火車站站下車。
    搭乘高鐵 → 在機場直接搭乘統聯巴士705前往高鐵桃園站，轉乘台灣高鐵至台中烏日站下車。
  • 從台中機場出發：
    搭乘巴士 → 搭9號公車至西勢十巷口站下車，至光大里站搭乘56號公車，至五權西忠明南路口下車。
    搭乘TAXI → 從台中機場至搭乘TAXI至一米陽光，車資約450元左右。

搭乘高鐵:

台北至台中烏日最快49分鐘;高雄到台中烏日最快42分鐘 → 下車後依照站內標示往車站1F就可以看到公車站牌，亦可搭計程車。

騎乘U-Bike:

從一米陽光步行3分鐘可抵達台中美術館U-Bike站。
